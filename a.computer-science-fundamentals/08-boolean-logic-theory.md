# Boolean Logic (Theory)

## Overview

Boolean logic is a branch of mathematics and computer science that deals with values that are either **True** or **False**. It forms the foundation of decision-making in programming and allows computers to evaluate conditions, compare values, and choose what action to perform.

Nearly every computer program uses Boolean logic to make decisions.

---

## Why It Matters

Boolean logic is important because it:

- Allows computers to make decisions
- Powers conditional statements like `if` and `else`
- Helps validate user input
- Controls loops and program flow
- Forms the basis of digital circuits and computer hardware

Without Boolean logic, computers would not be able to make decisions.

---

## What Is a Boolean Value?

A Boolean value can only have one of two possible states:

- **True**
- **False**

There are no other Boolean values.

**Examples:**

- The sky is blue → **True**
- 10 is greater than 20 → **False**
- A user is logged in → **True**
- A password is correct → **True**
- A file exists → **False**

---

## Boolean Expressions

A Boolean expression is a statement that evaluates to either **True** or **False**.

Examples:

- 8 > 5 → **True**
- 10 < 3 → **False**
- 15 == 15 → **True**
- 20 != 20 → **False**

Computers constantly evaluate these expressions while running programs.

---

## Comparison Operators

Comparison operators compare two values and return either **True** or **False**.

| Operator | Meaning | Example |
|----------|---------|---------|
| == | Equal to | 5 == 5 → True |
| != | Not equal to | 5 != 3 → True |
| > | Greater than | 10 > 7 → True |
| < | Less than | 2 < 8 → True |
| >= | Greater than or equal to | 10 >= 10 → True |
| <= | Less than or equal to | 6 <= 9 → True |

These operators are used in almost every programming language.

---

## Boolean Variables

A Boolean variable stores either **True** or **False**.

Examples:

- `isLoggedIn = True`
- `isAdmin = False`
- `hasPermission = True`
- `gameOver = False`

These variables help programs keep track of different states.

---

## Real-World Examples

Boolean logic appears in many everyday situations.

### Login System

If the password is correct:

→ Login succeeds (**True**)

Otherwise:

→ Login fails (**False**)

---

### ATM Machine

If the account balance is enough:

→ Dispense cash (**True**)

Otherwise:

→ Show "Insufficient Funds" (**False**)

---

### Online Shopping

If payment is successful:

→ Confirm the order (**True**)

Otherwise:

→ Cancel the purchase (**False**)

---

## Boolean Logic in Programming

Programs constantly ask questions such as:

- Is the user logged in?
- Is the password correct?
- Does the file exist?
- Is the player alive?
- Has the payment been completed?

Each question has only two possible answers:

- **True**
- **False**

The program then decides what to do based on the result.

---

## Boolean Logic in Computer Hardware

Boolean logic is not only used in programming—it is also the foundation of computer hardware.

Inside a computer, billions of tiny electronic switches called **transistors** represent:

- **1 (True / On)**
- **0 (False / Off)**

These tiny on/off states allow processors to perform calculations, store data, and execute instructions.

---

## Common Mistakes Beginners Make

New programmers often:

- Confuse **=` with `==`
- Assume Boolean values can have more than two states
- Forget that every comparison returns either **True** or **False**
- Write conditions that can never be true

Understanding Boolean logic helps avoid these mistakes.

---

## Key Takeaways

- Boolean logic works with only two values: **True** and **False**.
- Boolean expressions always evaluate to one of these two values.
- Comparison operators produce Boolean results.
- Boolean logic allows programs to make decisions.
- It is fundamental to both software and computer hardware.

---

## Summary

Boolean logic is one of the most fundamental concepts in computer science. It allows computers to evaluate conditions, make decisions, and control the flow of programs using only two possible values: **True** and **False**. Understanding Boolean logic is essential before learning conditional statements, loops, and more advanced programming concepts.
