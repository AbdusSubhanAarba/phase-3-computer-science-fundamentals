# phase-3-computer-science-fundamentals

This repository documents **Phase 3** of my structured Software Engineering learning journey, focusing on the fundamental concepts of computer science.

The goal of this phase is to build the logical thinking and problem-solving skills required to become a software engineer. Before writing code, it is important to understand how to analyze problems, design solutions, think algorithmically, and understand the mathematical foundations of computing.

This phase bridges the gap between understanding computers and learning how to solve problems like a programmer. It also prepares the foundation for future topics such as data structures, algorithms, databases, and software development.

---

## Topics Covered

- Introduction to Computational Thinking
- Problem-Solving Mindset
- Logical Thinking
- Problem Decomposition
- Algorithmic Thinking
- Pseudocode
- Flowcharts
- Boolean Logic (Theory)
- Boolean Logic (Applied)
- Discrete Math Basics
- Sets and Relations
- Binary and Hexadecimal Math
